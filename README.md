# ☯ 周易 · 三枚铜钱

> 在线摇卦工具 · 纯静态 · 开箱即用

三枚铜钱，六次投掷。自动排盘、卦辞详解、高岛易断、纳甲排盘一应俱全。  
**内置八字推算**，摇卦时自动记录时间八字，纳甲排盘同步带出。

---

## ✨ 功能

- 🪙 **摇卦** — 模拟三枚铜钱六次投掷，动爻高亮，自动生成本卦 / 变卦
- 📖 **卦辞详解** — 每卦的卦辞、彖传、象传、六爻爻辞
- 📜 **高岛易断** — 完整收录高岛嘉右卫门断辞，动爻逐爻解读，六爻皆静亦有专文
- 🔮 **纳甲排盘** — 为每爻配地支、五行、六亲（父母/兄弟/官鬼/妻财/子孙），标出世爻应爻
- ⏱ **实时八字** — 页面顶部实时显示当前时间及八字四柱
- 🕰️ **占卜时间戳** — 纳甲排盘弹窗及复制文本中带出摇卦当时的时间与八字
- 💾 **保存历史** — 最近 50 条摇卦记录（localStorage），含时间+八字，点击恢复
- 🖼️ **保存图片** — 将卦象导出为 PNG 图片
- 🔗 **URL 分享** — 通过 hash 参数分享卦象
- 🤖 **AI 友好复制** — 一键复制解读文本或纳甲排盘给 AI 使用

## 🛠️ 技术栈

纯前端静态项目，无需构建、无需服务端：

```
HTML5 + CSS3 + Vanilla JS
```

所有数据（64 卦、爻辞、高岛易断、纳甲数据）和逻辑（摇卦、排盘、八字推算）都在两个 JS 文件中，加载即用。

> 八字推算基于四柱法：年柱（干支取余）、月柱（节气定支 + 五虎遁）、日柱（2000-01-01 戊午为基准）、时柱（五鼠遁）。

## 🚀 部署

### GitHub Pages

```bash
git clone <你的仓库>
cd Yiin
git push origin main
```

在仓库 Settings → Pages → 选择 `main` 分支 → 保存即上线。

### 自定义域名

1. DNS 添加 `CNAME` 记录指向 `<你的用户名>.github.io`
2. 仓库 Settings → Pages → Custom domain 填入域名
3. （可选）根目录放一个 `CNAME` 文件防止部署后丢失

### 任意静态托管

本项目是纯静态文件，复制 `index.html` + `css/` + `js/` 到任何 HTTP 服务器即可运行。
