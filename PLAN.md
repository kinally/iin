# 《周易 · 三枚铜钱》项目说明

## 项目结构

```
Yiin/
├── yi-jing.html          ← 主页面（骨架）
├── css/
│   └── style.css         ← 全部样式
├── js/
│   ├── data.js           ← 八卦数据 + 64 卦表 + 高岛易断数据
│   └── app.js            ← 全部交互逻辑
├── PLAN.md
└── 计算器.xlsx            ← 原始数据源
```

## 已完成功能 ✅

### 核心
- [x] 64 卦名映射（`HEX` 数组）
- [x] 与 Excel 数据交叉验证，修复了 `谦`/`剥` 位置颠倒
- [x] 完整 UI：摇卦动画、本卦/变卦排盘、动爻高亮、解读文字

### 爻象
- [x] **全名显示** — "火风鼎""天水讼"格式，纯卦用"乾为天"
- [x] **分字配色** — 卦名各字按八卦元素着色
- [x] **爻值修复** — `judge()` 公式、标签同步
- [x] **阴爻样式** — `::before`/`::after` 两短杠经典样式
- [x] **元素配色** — 阳爻按上/下卦五行着色

### 详情
- [x] **卦辞详情页** — 含卦辞、彖传、象传、爻辞
- [x] **高岛易断** — 64 卦完整收录高岛嘉右卫门断辞，含动爻逐爻解读、六爻皆静时专文

### 交互
- [x] **保存此卦** — localStorage 存储最近 50 条
- [x] **分享此卦** — URL hash 链接，复制剪贴板
- [x] **历史记录** — 查看/点击恢复/清空

### 架构
- [x] 多文件拆分（html + css + js/data + js/app）
- [x] 加载顺序：`data.js` 先于 `app.js`，数据与逻辑分离

## 部署
- 可直接部署至 Cloudflare Pages（单目录，改一下 html 的 css/js 路径即可）

## 数据来源
- 卦辞、彖传、象传、爻辞原文引自《周易》
- 断辞部分节选自《高岛易断》（高岛嘉右卫门 著），每卦含断辞正文 + 六爻逐动解读
